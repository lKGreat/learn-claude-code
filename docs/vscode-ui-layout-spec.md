## 1.1 VS Code 三栏/多面板完整结构

VS Code 采用**活动栏 + 侧边栏 + 编辑区**的横向三栏，外加可选的**底部面板**，形成完整工作台：

```
┌────────────────────────────────────────────────────────────────────────────┐
│ 标题栏 │ 菜单栏                                                              │
├─┬────┬────────────────────────────────────────────────────────────────────┤
│ │侧边栏│  编辑区（可拆分为多个 EditorGroup）                                 │
│活│    │  ┌───────────────────────────────────────────────────────────────┐ │
│动│资源│  │ Tab1 | Tab2 | Tab3 ● | +   ← 多 Tab 标签栏                      │ │
│栏│管理│  ├───────────────────────────────────────────────────────────────┤ │
│ │器  │  │                                                                  │ │
│ │大纲│  │  主编辑区 / 拆分编辑区 / 预览区                                   │ │
│ │时间│  │                                                                  │ │
│ │线… │  └───────────────────────────────────────────────────────────────┘ │
├─┴────┴────────────────────────────────────────────────────────────────────┤
│ 底部面板（Panel，可折叠）： 终端 | 输出 | 调试控制台 | 问题 | ...            │
├────────────────────────────────────────────────────────────────────────────┤
│ 状态栏                                                                      │
└────────────────────────────────────────────────────────────────────────────┘
```

- **活动栏**：最左窄条，切换侧边栏视图（资源管理器、搜索、源代码管理等）
- **侧边栏**：根据活动栏选中显示不同面板（资源管理器、大纲、时间线等可堆叠或切换）
- **编辑区**：多 Tab、可拆分、支持预览
- **底部面板**：与编辑区共享垂直空间，可调整高度或隐藏

### 5.3 多 Tab 构造（EditorGroup）

主编辑区由 **EditorGroup**（编辑器组）管理，每组内为多 Tab 标签栏：

#### Tab 栏结构

```
┌─────────────────────────────────────────────────────────────────────────┐
│ [文件1] [文件2] [文件3 ●] [未保存.md ●] [预览.md] [+ 新编辑]             │
└─────────────────────────────────────────────────────────────────────────┘
```

- **Tab 项**：文件名、图标、关闭按钮 `×`
- **修改态**：未保存时 Tab 标题后显示 `●` 或标题加粗
- **预览态**：单击打开的文件 Tab 为*斜体*，再点其他文件会替换该 Tab，双击可固定
- **右侧**：`+` 新建编辑器按钮
- **溢出**：Tab 过多时横向滚动或下拉菜单

#### 拆分编辑器（多 EditorGroup）

- **水平拆分**：左右分栏，每栏独立 Tab 组
- **垂直拆分**：上下分栏
- **触发**：标题栏「拆分编辑器」图标、右键 Tab「向右拆分」、拖拽 Tab 到边缘

```
┌───────────────────────┬───────────────────────┐
│ Tab A | Tab B | +     │ Tab C | Tab D | +     │
├───────────────────────┼───────────────────────┤
│                       │                       │
│    EditorGroup 1      │    EditorGroup 2      │
│                       │                       │
└───────────────────────┴───────────────────────┘
```

### 5.4 预览文件（Preview）

#### 单击预览（singleClickPreview）

- **行为**：在文件树单击文件 → 以预览模式打开，占一个 Tab
- **预览 Tab**：斜体显示，无固定状态
- **替换**：再单击其他文件 → 替换当前预览 Tab
- **固定**：双击文件、或编辑内容、或点击 Tab 上的「保持打开」→ 转为正式 Tab

#### Markdown / 图片等预览

- **Markdown**：「打开预览」→ 侧边或新 Tab 显示渲染结果，可与源文件并排
- **图片**：直接预览，无需编辑器
- **其他**：根据文件类型由扩展提供预览
